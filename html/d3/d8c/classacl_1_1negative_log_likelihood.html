Detailed Description

Base class for negative loglikelihoods used in composition data.

This class has two virtual methods: nloglike and residual.

void acl::negativeLogLikelihood::tail_compression ( )

Determine non-zero array elements for tail compression.

This routine fills the member variables m_jmin and m_jmax with the array indexes that correpsond to the cumulative sum that is greater than pmin, and 1-pmin respectively. This is then later used to construct ragged objects to be used in the likelihood functions.
